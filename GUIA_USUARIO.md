# 🚀 Trend Hub - Guia Rápido de Uso

## Início Rápido

### 1. Login/Registro
- Acesse a página de autenticação
- Crie uma conta com email, senha, username único e nome completo
- Ou faça login se já tem conta

### 2. Completar Perfil
- Clique em seu avatar no canto inferior esquerdo
- Edite seu nome, bio e foto de perfil
- As alterações são salvas automaticamente

---

## 📝 Criar e Gerenciar Publicações

### Como Publicar Texto
1. No Feed, clique no campo "O que está acontecendo?"
2. Digite seu texto (até 1000 caracteres)
3. (Opcional) Escolha uma comunidade no seletor
4. Clique "Publicar"

### Como Publicar Foto
1. Click no campo de post
2. Clique no botão "Imagem" 
3. Cole a URL da imagem
4. Veja o preview
5. Clique "Publicar"

### Como Publicar Vídeo
1. Click no campo de post
2. Clique no botão "Vídeo"
3. Cole a URL (YouTube, Vimeo, etc)
4. Clique "Publicar"

### Gerenciar Post
- **Curtir**: Clique no ❤️ para curtir/descurtir
- **Comentar**: Clique no 💬 para abrir painel de comentários
- **Compartilhar**: Clique em 🔗 para copiar link
- **Deletar**: Menu (•••) → Excluir (só aparece em seus posts)

---

## 💬 Comentários

### Comentar em um Post
1. Clique no botão de comentário (💬)
2. O painel de comentários abre
3. Digite seu comentário (até 500 caracteres)
4. Clique em "Send" (avião)

### Ver Comentários
- Todos os comentários aparecem em ordem cronológica
- Cada comentário mostra: avatar, nome, @username, tempo e conteúdo

### Deletar Comentário
- Você só pode deletar seus próprios comentários
- Botão de lixeira aparece ao lado do comentário

---

## 👥 Sistema de Relacionamentos

### Seguir Usuários

**Opção 1: No Sidebar Direito**
- RightSidebar mostra "Sugestões" 
- Clique "Seguir" rapidamente

**Opção 2: No Perfil do Usuário**
1. Clique no post de alguém
2. Clique no avatar/nome do autor
3. Clique no botão "Seguir"

**Opção 3: Explorar**
1. Vá para "Explorar" (ou barre com 🔍)
2. Busque usuário por nome/username
3. Clique no perfil → "Seguir"

### Ver Seguidores/Seguindo
- Clique em seu perfil (último ícone na sidebar)
- Veja estatísticas: Posts, Seguidores, Seguindo

### Deseguir
- Vá para perfil do usuário
- Clique no botão "Seguindo" para deseguir

---

## 💌 Mensagens Privadas

### Iniciar Conversa
1. Vá para perfil de um usuário
2. Clique no botão "Mensagem"
3. A conversa abre automaticamente

**Ou:**
1. Vá para "Mensagens" (ícone 💬 na sidebar)
2. Procure usuário usando a busca (🔍)
3. Clique no usuário para iniciar conversa

### Enviar Mensagem
1. Digite sua mensagem
2. Pressione Enter ou clique no botão "Send"
3. Mensagens aparecem em tempo real

### Ver Conversas
- "Mensagens" mostra lista de todas suas conversas
- Ordenadas por última mensagem
- Clique em uma para abrir o chat

---

## 👥 Comunidades

### Participar de Comunidade
1. Vá para "Comunidades" (sidebar)
2. Procure a comunidade que gosta
3. Clique "Participar"

### Publicar em Comunidade
1. Crie um novo post
2. No campo de post, abra o seletor de comunidade
3. Escolha a comunidade
4. Publique como normal

### Ver Posts da Comunidade
1. Vá para "Comunidades"
2. Clique na comunidade
3. Veja todos os posts dela

### Criar Comunidade
1. No Sidebar Esquerdo, vá para "Minhas Comunidades"
2. Clique no botão "+" (mais)
3. Preencha: título, descrição, categoria, regras
4. Pronto! Você é o criador

---

## 🔍 Explorar & Descobrir

### Buscar Usuários
- Clique na barra de busca (topo do sidebar)
- Digite nome ou @username
- Clique no perfil para visitar

### Ver Trending
- Vá para "Trends" na sidebar
- Veja tags em alta com número de posts
- Clique em uma tag para ver posts relacionados

### Sugestões
- RightSidebar mostra usuários sugeridos
- RightSidebar mostra comunidades em destaque

---

## 👤 Perfil Pessoal

### Editar Perfil
1. Vá para seu Perfil (último ícone)
2. Clique "Editar perfil"
3. Mude nome, bio
4. Clique "Salvar"

### Mudar Foto de Perfil
1. Vá para seu Perfil
2. Clique na câmera (📷) no canto inferior do avatar
3. Escolha arquivo de imagem
4. Upload automático

### Ver Estatísticas
- Número de posts
- Número de seguidores
- Número de pessoas que você segue
- Data de criação da conta

---

## ⚙️ Configurações & Logout

### Sair da Conta
1. No Sidebar inferior, clique no botão "Sair" (🚪)
2. Você será redirecionado para login

## ❓ Problemas Comuns

**P: Não vejo posts no meu feed?**
- Você não está seguindo ninguém ainda
- Vá para "Explorar" ou "Comunidades"
- Siga usuários e participe de comunidades

**P: Não consigo enviar mensagem?**
- Certifique-se de estar logado
- Tente recarregar a página

**P: Foto não aparece no post?**
- Certifique-se que a URL é pública e válida
- Tente outra URL de imagem

**P: Não consigo comentar?**
- Certifique-se de estar logado
- Tente recarregar a página
